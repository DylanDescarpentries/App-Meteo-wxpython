Français:
Projet d'exercice perso pour s'entraîner sur le module wxpython. Certaine parti du code sont à modifier :
logo : à supprimer ou a ajouter votre logo
api_key : s'inscrire sur https://openweathermap.org/ et réclamer votre api_key gratuite (ou payante)/
La langue tts = gTTS(texte_audio, lang='fr') remplacer 'fr' par votre langue.

English:
Personal exercise project to test using the wxPython module. Some parts of the code need to be modified:
logo: remove or add your logo
api_key: sign up at https://openweathermap.org/ and claim your free (or paid) API key.
The language tts = gTTS(text_audio, lang='fr') replace 'fr' with your language.
